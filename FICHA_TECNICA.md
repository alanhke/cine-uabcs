# Resumen Técnico - CineUABCS

- **Framework:** Next.js 14 (App Router)
- **Lenguaje:** TypeScript
- **Base de Datos:** MySQL (Relacional)
- **ORM:** Prisma
- **Autenticación:** NextAuth.js (Gestión de sesiones de cliente y admin)
- **Estilos:** Tailwind CSS (Arquitectura Mobile-First)
- **APIs Externas:** TMDB API (Información de estrenos en tiempo real)
- **Librerías Clave:** 
  - `html2canvas`: Generación de imágenes para el módulo Wrapped.
  - `recharts`: Visualización de datos en Dashboard Administrativo.
  - `Lucide React`: Iconografía técnica.
- **Gestión de Archivos:** Sistema local de File System para carga de pósters en `public/uploads`.
